# Arquitetura do Sistema - Vacina Kids

O projeto segue uma arquitetura modularizada, baseada em Spring Boot, com separação de responsabilidades (Controller -> Service -> Repository).

## Diretórios e Estrutura
- **frontend/**: Código da aplicação Angular/Ionic.
- **backend/**: Código da aplicação Spring Boot.
- **docs/**: Documentações do projeto (arquitetura, banco de dados, API).
